# Cómo procesar órdenes

Guía para revisar, preparar y cerrar órdenes en el panel.

## Cuándo usar esta guía

Usa esta guía cuando llegue una orden de:

- Tienda online.
- Venta capturada manualmente.
- WhatsApp registrada en el sistema.
- Marketplace integrado o conciliado manualmente.

## Estados comunes

- **Pendiente**: requiere revisión o acción.
- **Pagada**: pago confirmado.
- **Preparación**: se está separando mercancía.
- **Enviada**: ya tiene entrega o guía registrada.
- **Completada**: orden cerrada correctamente.
- **Cancelada**: no se surtirá.

Los nombres exactos pueden variar según configuración del panel.

## Pasos para procesar una orden

1. Entra al panel.
2. Abre el menú **Órdenes**.
3. Busca la orden por:
   - Número de orden.
   - Nombre del cliente.
   - Correo.
   - Fecha.
   - Estado.
4. Abre el detalle de la orden.
5. Revisa datos del cliente:
   - Nombre.
   - Teléfono.
   - Correo.
   - Dirección de envío.
6. Revisa productos:
   - Modelo.
   - Talla.
   - Color.
   - Cantidad.
7. Verifica pago:
   - Método de pago.
   - Estado del pago.
   - Monto total.
8. Si el pago está confirmado, prepara la orden.
9. Registra envío o entrega:
   - Paquetería o chofer.
   - Número de guía, si aplica.
   - Fecha de salida.
10. Marca la orden con el estado correspondiente.
11. Confirma que el inventario se haya descontado o reservado correctamente.

## Orden manual

Para ventas por WhatsApp o pedidos especiales:

1. Abre **Órdenes**.
2. Selecciona opción para crear orden manual o borrador.
3. Agrega cliente.
4. Agrega productos y variantes exactas.
5. Revisa cantidades y precios.
6. Captura método de pago o nota de pago pendiente.
7. Guarda la orden.
8. Avanza el estado cuando pago y entrega estén confirmados.

## Cancelaciones

Antes de cancelar:

- Confirmar con el responsable de venta.
- Revisar si ya hubo pago.
- Revisar si ya se preparó o envió mercancía.
- Confirmar si el inventario debe regresar a disponible.

## Verificación

Antes de cerrar una orden:

- El pago está correcto.
- Las tallas y colores coinciden con lo solicitado.
- La dirección está completa.
- El envío o entrega quedó registrado.
- El cliente recibió confirmación, si aplica.

## Errores comunes

- Procesar orden sin pago confirmado.
- Enviar talla equivocada.
- No registrar número de guía.
- No actualizar estado después de entregar.
- No revisar órdenes pendientes del día.
