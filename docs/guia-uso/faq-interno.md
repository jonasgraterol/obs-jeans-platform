# FAQ interno

Preguntas frecuentes para el equipo de OBS Jeans.

## ¿Dónde entro al panel?

- Producción: https://admin.jeansobs.com/app
- Staging / pruebas: https://admin-staging.jeansobs.com/app

## ¿Cuándo uso producción y cuándo staging?

- Producción: pedidos, inventario, clientes y ventas reales.
- Staging: pruebas, capacitación y validaciones antes de tocar datos reales.

## ¿Qué hago si una talla no aparece?

1. Buscar el producto.
2. Abrir variantes.
3. Confirmar si la talla fue creada.
4. Si no existe, crear variante solo si está autorizada.
5. Si sí existe, revisar si tiene inventario asignado.

## ¿Qué hago si una variante aparece sin stock pero físicamente sí hay piezas?

1. Confirmar producto, talla y color.
2. Revisar si hay stock reservado por órdenes.
3. Hacer conteo físico.
4. Ajustar inventario si la diferencia está confirmada.
5. Anotar motivo del ajuste.

## ¿Qué significa stock bajo?

Para OBS Jeans, stock bajo es cuando una variante tiene menos de 5 unidades disponibles.

## ¿Puedo modificar precios?

No modificar precios sin autorización explícita de administración o Jonas.

## ¿Puedo borrar un producto?

No borrar productos sin autorización. Si un producto ya no se vende, es mejor desactivarlo o dejarlo en estado no publicado, según lo que permita el panel.

## ¿Qué hago si una orden ya se pagó pero aparece pendiente?

1. Abrir la orden.
2. Revisar historial de pagos.
3. Confirmar comprobante o pasarela.
4. Si el pago fue manual, registrar nota interna.
5. Avisar al responsable antes de marcarla como pagada.

## ¿Qué hago si una clienta pide cambio de talla?

1. Buscar la orden original.
2. Confirmar producto y talla comprada.
3. Revisar inventario de la nueva talla.
4. Registrar el cambio siguiendo el flujo de cambios/devoluciones del panel.
5. Confirmar si hay diferencia de precio o envío.

## ¿Qué hago si una orden debe cancelarse?

1. Confirmar con el responsable.
2. Revisar si ya hubo pago.
3. Revisar si ya se envió.
4. Cancelar solo si corresponde.
5. Confirmar que el inventario regrese a disponible cuando aplique.

## ¿Cómo busco a una clienta?

1. Abrir **Clientes**.
2. Buscar por nombre, correo o teléfono si está disponible.
3. Revisar historial de órdenes.
4. No compartir datos personales fuera del equipo autorizado.

## ¿Qué datos necesito para crear un producto?

- Nombre.
- Descripción.
- Fotos.
- Tallas.
- Colores.
- SKUs.
- Códigos de barras.
- Precio autorizado.
- Stock inicial.

## ¿Qué hago si no estoy segura de un cambio?

1. No guardar el cambio todavía.
2. Tomar screenshot.
3. Compartirlo con Jonas o administración.
4. Esperar autorización.

## ¿Cómo reporto un problema del sistema?

Enviar al equipo técnico:

- Qué estabas intentando hacer.
- Link o pantalla donde pasó.
- Screenshot.
- Hora aproximada.
- Producto, orden o cliente afectado.
- Mensaje de error exacto, si apareció.
