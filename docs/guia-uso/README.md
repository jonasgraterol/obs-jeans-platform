## Ambientes

- Producción: https://admin.jeansobs.com/app
- Staging / pruebas: https://admin-staging.jeansobs.com/app

## Reglas rápidas

- Usar producción solo para operaciones reales.
- Usar staging para practicar, validar una guía o hacer pruebas.
- Antes de cambiar precios, productos o inventario real, confirmar que se está en el ambiente correcto.
- Mantener nombres, tallas, colores y SKUs consistentes con el catálogo interno de OBS Jeans.
- Considerar stock bajo cuando una variante tiene menos de 5 unidades.

## Flujo recomendado para el equipo

1. Iniciar sesión en el panel.
2. Confirmar ambiente: producción o staging.
3. Ir al módulo correspondiente: Productos, Inventario, Órdenes, Clientes, Promociones o Reportes/Exportaciones.
4. Hacer el cambio o consulta.
5. Verificar el resultado antes de cerrar la ventana.
6. Si el cambio afecta a clientas o marketplace, avisar al equipo responsable.
